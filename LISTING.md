Ready-to-paste copy for submitting joey-bot to bot listing sites (top.gg,
discordbotlist.com, discord.bots.gg, discords.com/bots). You handle the actual
submissions — these all require logging in with your Discord account.

## Core metadata

- **Bot name:** joey-bot
- **Prefix:** `/` (slash commands)
- **Invite link** (includes the scopes slash commands need):
  `https://discord.com/oauth2/authorize?client_id=1516042370439839784&scope=bot+applications.commands&permissions=0`
- **Suggested tags:** `Fun`, `Utility`, `Games`, `Meme`, `Social`, `Image`

> Before submitting: set the bot to **Public** in the Developer Portal (Bot tab),
> make sure it's in **at least one server**, and give it an avatar. Several sites
> also require you to add their stats-posting token or invite a verification bot.

## Per-site notes

| Site | Notes |
|---|---|
| **top.gg** | Paste short + long (Markdown). Pick up to a few tags. Requires you to be the bot owner & verify. |
| **discordbotlist.com** | Markdown long description supported; has separate "short" field (≤280). |
| **discord.bots.gg** | Requires short + long; bot must already be in a server. Manual approval. |
| **discords.com/bots** | Short + long Markdown; supports screenshots — add `/help` and a comic command screenshot. |

## Tips

- Grab a **screenshot or two** (the `/help` overview and a comic or `/pokemon` embed) — the
  bigger sites rank listings with images higher.
- Keep a one-line **"What's new"** handy to update as you add features.
